# mobile-edit-pro
Mobile Edit Pro is a modern, CapCut-inspired online video editor built with Next.js, React, Tailwind CSS, Firebase, and FFmpeg. Edit videos directly in your browser with a timeline, effects, transitions, text, audio tools, and cloud project saving. Deployed on Vercel for fast, responsive performance.
